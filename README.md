
🪑 Furni – Furniture Dashboard Interface
📌 Description

Furni is a modern, fully responsive Furniture Dashboard Interface designed to provide an intuitive and user-friendly experience for managing furniture-related data. The interface focuses on clean design, accessibility, and smooth interaction across all screen sizes.

🎯 Project Objective

The goal of this project is to design and develop a professional dashboard UI that can be used for managing furniture products, tracking orders, and visualizing basic data in a simple and attractive way.

✨ Features
Fully responsive design (Mobile, Tablet, Desktop)
Modern and clean user interface
Dashboard layout with summary cards
Product management section (UI)
Order tracking interface (UI)
Sidebar navigation menu
Search and filter UI components
Smooth and user-friendly interaction
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive design (Flexbox & Grid)
JavaScript (Vanilla JS) – Interactivity
